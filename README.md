# ğŸ“Š Numerical Methods for Machine Learning (MATLAB)

This project explores **core numerical methods behind modern machine learning** using the classic **Iris dataset**.  
All algorithms are implemented from a **mathematical and algorithmic perspective** using **MATLAB**.

Rather than relying on black-box ML libraries, this project demonstrates how:
- Data is projected  
- Clusters are formed  
- Latent factors are extracted  
- Class separation is maximized  

using **linear algebra and numerical optimization**.
